1. Date객체 생성

- new Date()
- new Date(987656789789) : 1970.01.01 기준으로 경과한 날짜
- new Date("2002.06.15")
- new Date(2002,06,15,12,13,52)
- 월은 0월부터 시작, 숫자로 Date 객체 생성시 하나 작은 값으로 설정한다


2. Date객체 생성

- now Date()
- now.toLocaleString() : 날짜와 시간
- now.toLocaleDateString() : 날짜
- now.toLocaleTimeString() : 시간
- 월은 0월부터 시작, 숫자로 Date 객체 생성시 하나 작은 값으로 설정한다


3. 요일구하기

- now = new Date()
- day = now.getDay() : 0(일요일) ~ 6(토요일)
